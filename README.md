# Sun Arpeggios SC

Live performance system in SuperCollider for split keyboard control with:

- low register drones / organ-like bass
- high register random arpeggios
- velocity-based dynamics
- MIDI control mapping for Arturia MiniLab 3
- scene presets:
  - darkAmbient
  - organHall
  - brightGlass
  - cinematicForte

## Requirements

- SuperCollider 3.13.0
- MIDI keyboard / controller
- Tested with Arturia MiniLab 3 on Windows

## Files

- `scripts/live_split_arp_scenes.scd` -> main performance script
- `scripts/cleanup.scd` -> stop and cleanup script

## Usage

1. Boot SuperCollider
2. Run the main script
3. Play low notes for bass / drone behavior
4. Play high notes for arpeggiated texture
5. Use MIDI controls to shape dynamics and effects
